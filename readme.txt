A plugin created to help our publishers integrate our adunits using music data.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add artist and song, or genre data when creating posts.
